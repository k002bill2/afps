# Resume

`/compact` 또는 `/clear` 후 이전 세션의 컨텍스트를 복원합니다.

## Instructions

다음 단계를 순서대로 실행하세요:

### Step 1: 활성 프로젝트 스캔

```bash
ls -la dev/active/*.md 2>/dev/null || echo "No active projects found"
```

### Step 2: 프로젝트 목록 표시

활성 프로젝트가 있으면 목록을 표시합니다:

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
ACTIVE PROJECTS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

[번호] [프로젝트명]
    Status: [상태]
    Updated: [마지막 업데이트]
    Progress: [진행률]

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### Step 3: 프로젝트 선택

- 프로젝트가 1개면 자동 선택
- 2개 이상이면 사용자에게 선택 요청

### Step 4: 컨텍스트 로드

선택된 프로젝트의 파일들을 읽습니다:
- `*-context.md`: 배경, 진행 상황, 세션 히스토리
- `*-tasks.md`: 할 일 목록, 우선순위

### Step 5: 컨텍스트 요약 출력

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
CONTEXT RESTORED: [프로젝트명]
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

## Overview
[프로젝트 개요]

## Current State
- Progress: [진행률]
- Last Session: [마지막 세션 요약]
- Blockers: [블로커 있으면 표시]

## Pending Tasks (Top 5)
1. [ ] [작업1]
2. [ ] [작업2]
...

## Ready to Continue
다음 작업을 시작하시겠습니까?

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### Step 6: TodoWrite로 할 일 복원

`*-tasks.md`에서 미완료 항목을 TodoWrite 도구로 등록합니다.

## Error Handling

프로젝트가 없는 경우:

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
NO ACTIVE PROJECTS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

dev/active/ 디렉토리에 활성 프로젝트가 없습니다.

새 프로젝트를 시작하려면:
1. 작업 내용을 설명해주세요
2. /save-and-compact로 진행 상황을 저장할 수 있습니다

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```
